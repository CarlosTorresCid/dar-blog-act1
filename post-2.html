CSS responsive sin frameworks

Publicado el 4 de noviembre de 2025 · CSS

Con unas pocas técnicas podemos cubrir la mayoría de casos sin depender de librerías pesadas: variables CSS, flexbox, grid y tamaños fluidos.

Buenas prácticas

  • Define variables para colores/espaciados y reutilízalas.
  • Usa grid/flex para componer tarjetas y columnas.
  • Limita el ancho de texto a ~70 caracteres para legibilidad.
  • Optimiza imágenes y añade loading="lazy".

Los @media en nuestro proyecto ajustan el número de columnas y el comportamiento del menú para pantallas pequeñas.
